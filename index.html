User Interface Tests to Remote Control Software

This project arises because Aranda offers a technical assistance product with remote support for its customers, in real time and from anywhere in the world. This product requires the implementation of tests since there is not enough time and personnel to perform them manually.

Carrying out manual tests has a very high in terms of both time and personnel.

Our solution to this problem was to ally ourselves with automation that helps us to reduce repetitive or complex tasks that require human effort. One of the tools that helped us to fulfill this purpose is Selenium that allows us to record, edit, and debug cases that can be executed automatically and can be replicated later, and even with a scalable functionality to cover a simulation of many users at the same time.

Want to know how we did the tests?

Here is a video of an automated test where you can see how, after a single click, the program automatically starts running the test by entering the browser, searches for the web page, enters the user and password and moves to the main screen where the device management is performed.

A total of 42 tests were performed, which would have taken a single person 8 hours to perform
and the programmed machine takes only 20 minutes.
